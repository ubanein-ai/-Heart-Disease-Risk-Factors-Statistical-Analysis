1.OVERVIEW

This project provides a comprehensive analysis of cardiovascular health risks using Python. It integrates clinical patient data with large-scale health indicators to identify how variables like Cholesterol, Age, and Sex contribute to the likelihood of heart disease.

2.DATASET

-Datasetsheart.csv: A clinical dataset (303 records) focusing on biological markers such as Resting Blood Pressure, Maximum Heart Rate, and Thalassemia.

-Heart_disease_health_indicators.csv: A large survey dataset (250k+ records) that explores the intersection of lifestyle choices (Smoking, Physical Activity, Diet) and chronic health conditions.

3.KEY FEATURES

-Exploratory Data Analysis (EDA): Profiling patient demographics and health metrics using Pandas.

-Statistical Validation: Applying the Chi-Square Test of Association to determine if there is a statistically significant link between categorical health indicators.

-Data Visualization: Using Seaborn and Matplotlib to visualize risk distributions and correlation heatmaps.

4.HYPHOTESIS SETTINGS:

-H_0 (Null Hypothesis): There is no association between the health factor and heart disease.

-H_1 (Alternative Hypothesis): There is a significant association between the health factor and heart disease.

5.CONCLUSION
The results of this study aim to highlight the most critical "red flag" health indicators to assist in early heart disease screening and prevention
